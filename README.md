عربي بالكامل (RTL)، ثابت Vanilla (HTML/CSS/JS) بدون أي اعتماديات ثقيلة، جاهز لـ GitHub Pages.

## مصدر الصور
صور حقيقية من صفحة المطعم على **خرائط قوقل**، وقد تم **تحسينها لونيًا** مسبقًا (color-grading).
تمت تنقيتها يدويًا: استُبعدت لوحات الأسعار، والصور المكرّرة/غير الواضحة، وصور **مطعم مجاور مختلف**
(«النور الشامي») التي لا تخص أوركي جريل.

## الموشن التوقيعي والحياة المحيطة
- **التوقيعي:** سيخ كباب SVG يدور (scaleX wobble) مع توهّج فحم دافئ، نقشة إزنيق تُرسم
  (stroke-dashoffset)، وبخار متصاعد.
- **الحياة المحيطة:** بارالاكس خفيف (≤12px، معطّل على اللمس)، توهّج «يتنفّس»، لمعة sheen على الأزرار.
- **كوريغرافيا السكول:** ظهور اتجاهي متدرّج (spring ease)، رسم المونوغرام عند التحميل، count-up للتقييم.
- كل الحركة `transform/opacity/filter` فقط، وتُلغى بالكامل عند `prefers-reduced-motion`.
